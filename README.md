# moctale-ui-clone
A modern, responsive movie discovery web application inspired by contemporary movie-tracking platforms. Built with **React**, **Vite**, and **Tailwind CSS**, this project focuses on delivering a clean, dark-themed user interface with smooth animations and reusable components.

> ⚠️ **Note:** This is an original frontend implementation inspired by modern movie discovery apps. It does not use or include any proprietary source code, assets, or backend from other platforms.

---

## ✨ Features

- 🏠 Beautiful Home Page
- 🔍 Advanced Search UI
- 🎬 Explore Movies
- 📚 Collections
- 👤 User Profile
- 🎞️ Movie Details Page
- 🌙 AMOLED Dark Theme
- 📱 Fully Responsive Design
- ⚡ Smooth Animations
- ❤️ Watch Later UI
- ⭐ Trending & Most Interested Sections
- 🎨 Modern Card-Based Layout
- 🧩 Reusable Components

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/movieverse-ui.git
```

### Navigate to the project

```bash
cd movieverse-ui
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

The application will run on:

```
http://localhost:5173
```
